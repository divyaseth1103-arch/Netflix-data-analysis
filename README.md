# 🎬 Netflix Data Analysis

## About
Analysed 8000+ Netflix titles using Python, Pandas and Matplotlib.

## Key Findings
- 69.7% of content is Movies, 30.3% TV Shows
- USA has the most content (2800+ titles), India is 2nd
- International Movies & Dramas are the top genres
- Rajiv Chilaka is the most prolific director with 19 titles

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
Kaggle — Netflix Movies and TV Shows
